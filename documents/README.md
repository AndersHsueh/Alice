## 💡 使用技巧

### Obsidian 功能

1. **图谱视图**: 查看文档关系网络
2. **搜索**: 全文搜索 + 标签搜索
3. **反向链接**: 查看被引用情况
4. **Canvas**: 可视化架构图
5. **Dataview**: 动态查询 (需插件)

### 快捷键 (Obsidian)

| 功能 | 快捷键 (Windows) | 快捷键 (Mac) |
|------|-----------------|-------------|
| 快速打开 | `Ctrl+O` | `Cmd+O` |
| 全局搜索 | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| 图谱视图 | `Ctrl+G` | `Cmd+G` |
| 新建笔记 | `Ctrl+N` | `Cmd+N` |
